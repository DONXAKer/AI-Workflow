ПРОБЛЕМЫ И РЕШЕНИЯ В ХОДЕ СЕССИИ
----------------------------------------

1. CSRF (403 Forbidden на POST /api/runs)
   Проблема: Spring Security 6 использует XOR-токены, curl отправляет raw cookie
   Решение: Заменили XorCsrfTokenRequestAttributeHandler на CsrfTokenRequestAttributeHandler

2. Модель z-ai/glm-5.1 зависла
   Проблема: Модель не отвечала 30+ минут (вероятно нет tool-use поддержки)
   Решение: Переключились на deepseek/deepseek-chat-v3-0324

3. working_dir не установлен в shell_exec блоках
   Проблема: ShellExecBlock требует working_dir из конфига или проекта
   Решение: Добавили working_dir: /warcard во все shell_exec блоки feature.yaml

4. git не установлен в контейнере
   Проблема: eclipse-temurin:21-jre-alpine не содержит git
   Решение: Коммит выполнить вручную на хосте

5. Gradle cache deadlock при пересборке
   Проблема: Параллельные docker-контейнеры захватывали gradle lock файлы
   Решение: Удаление *.lock файлов в gradle-cache volume

6. CLI approval gate бесконечный цикл
   Проблема: CliApprovalGate читает stdin, в Docker без TTY → бесконечный цикл
   Решение: Установили approval: false во всех блоках feature.yaml
